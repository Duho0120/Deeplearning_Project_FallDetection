## 기술적 고려사항

### 1. 데이터셋 선정
- **후보 데이터셋**:
  - UR Fall Detection Dataset
  - Multiple Cameras Fall Dataset
  - Fall Detection Dataset (Kaggle)
  - Le2i Fall Detection Dataset
  
### 2. 모델 아키텍처 후보
- **비전 기반**:
  - CNN (Convolutional Neural Networks)
  - 3D CNN (시공간 특징 추출)
  - Two-Stream Networks
  
- **시퀀스 기반**:
  - LSTM (Long Short-Term Memory)
  - GRU (Gated Recurrent Unit)
  
- **최신 아키텍처**:
  - Transformer 기반 모델
  - Pose Estimation + Classification (MediaPipe, OpenPose)

### 3. 성능 목표
- **정확도(Accuracy)**: 95% 이상
- **재현율(Recall)**: 높은 우선순위 (낙상 놓치지 않기)
- **정밀도(Precision)**: 균형 있는 수준
- **추론 시간**: 실시간 처리 가능 (<100ms)
